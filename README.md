# Vegetable Tic-Tac-Toe

A fun, React-based twist on the classic Tic-Tac-Toe game — using vegetables instead of X’s and O’s.  
This project was created to practice React components, props, state management, and reusable UI sections.

---

## Features
- Classic Tic-Tac-Toe gameplay  
- Vegetable icons instead of X/O  
- Built entirely with React  
- Reusable component structure  
- Simple, clean, and responsive UI

## Installation and Setup

Clone the repository:

git clone https://github.com/jennessa-jv/Vegetable-tic-tac-toe

Enter the project folder:

cd Vegetable-tic-tac-toe

Install dependencies:

npm install

Run the app:

npm start

Open the app in your browser:

http://localhost:3000/

---

## How to Play
1. Click any empty square  
2. Player 1 uses one vegetable icon  
3. Player 2 uses another vegetable icon  
4. First player to get 3 in a row wins  
5. Refresh to restart (or build your own Reset button)
